# Botão

O componente `ButtonComponent` é usado para ações clicáveis no sistema. Ele pode ser utilizado em diversas variações, como botão padrão, contornado, com ícones, etc.

## Exemplos de uso

## Quando usar

- Para representar ações primárias em formulários ou modais.
- Quando houver necessidade de chamar atenção para uma ação importante.
- Use a versão **contornada** para ações secundárias.

---

## Boas práticas

- Mantenha o texto curto e claro.
- Use apenas um botão principal por tela.
- Combine com ícones apenas quando necessário para reforçar o significado da ação.
